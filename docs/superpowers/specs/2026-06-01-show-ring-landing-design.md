# Дизайн: лендинг «Show Ring»

**Дата:** 2026-06-01 · **Статус:** утверждён к реализации

## Контекст
Конвертящий одностраничный лендинг — «визитная карточка» платформы Show Ring (выставки животных, РКФ/FCI). ЦА — заводчики, участники, организаторы (трафик РФ, основной — мобильный). Строится в текущем стеке (Next.js 16, MUI 7, Minimal Kit) с максимальным переиспользованием готовых компонентов.

## Решения (согласованы с пользователем)
- **Размещение:** заменяет главную `/` (`src/app/(home)/page.tsx` → новый `LandingView`). Демо Minimal (`HomeView`) остаётся в репозитории, но не на `/`.
- **Двуязычность RU+EN:** самодостаточный словарь `src/sections/landing/content/` (ключи `{ en, ru }`) + лёгкий переключатель RU/EN на лендинге. Глобальная i18n-конфигурация (где `ru` нет) НЕ трогается — во избежание скоупа на весь дашборд. Дефолт — RU.
- **Декор:** генерируемый сет тематических **SVG** (`public/assets/landing/`): ринг/подиум, наградная розетка, кубок, лапа, силуэты собаки и кошки, кольцо-ринг, плюс волны/блобы. Фотореалистичные растры не генерируются. Фото-фоны (hero/FAQ/новости) — существующие `public/assets/background/*.webp`.
- **CTA:** основная «Войти в ринг» / «Регистрация» → `/auth/jwt/sign-up`; вторичная «Войти» → `/auth/jwt/sign-in`.

## Структура (секция ТЗ → переиспользование)
1. **Hero** — H1 (УТП), подзаголовок, 2 CTA, фон webp + SVG-кольца/розетка, `motion` (varFade). Паттерн `home-hero`/`hero-background`.
2. **Преимущества** — сетка 4 карточек с `Iconify`-иконками. Паттерн `home-minimal`/`home-highlight-features`.
3. **Статистика** — 3–4 крупные цифры с `AnimateCountUp`, анимация при попадании во вьюпорт (`MotionViewport`/`InView`).
4. **Roadmap** — 3 шага, `@mui/lab` Timeline (есть в проекте) с иконками шагов.
5. **FAQ** — аккордеон (паттерн `home-faqs`) поверх `background-*.webp` с затемняющим overlay для читаемости.
6. **Новости/события** — слайдер карточек на `carousel` (embla): фото (webp-плейсхолдер) + заголовок + дата + краткий итог; фокус на выставках РФ.
7. **SEO-префутер** — читаемый текстовый блок с ключами (выставки собак/кошек, Show Ring, регистрация на выставку, календарь выставок РФ).

## Файловая структура
- `src/sections/landing/content/{en,ru}.ts` + `index.ts` (типизированный словарь), `landing-lang-context.tsx` (контекст языка + хук + переключатель).
- `src/sections/landing/landing-hero.tsx`, `-advantages.tsx`, `-stats.tsx`, `-roadmap.tsx`, `-faqs.tsx`, `-news.tsx`, `-seo.tsx`.
- `src/sections/landing/view/landing-view.tsx` + `index.ts`.
- `public/assets/landing/*.svg` — сет декора.
- `src/app/(home)/page.tsx` → рендерит `LandingView`, обновить metadata (title/description Show Ring, RU).

## Копирайтинг
Скилл «копирайтер» в среде недоступен — текст (УТП, подзаголовки, тезисы, FAQ, SEO) пишется вручную, RU + EN, тон: современный, доверительный, профессиональный.

## Нефункциональные
- Адаптив (mobile-first), быстрая загрузка (SVG-декор, webp, без тяжёлых скриптов), доверительный чистый стиль.
- Кириллица: рендерится системным фолбэком/Inter (Cyrillic) — приемлемо.

## Верификация
`npx tsc --noEmit` (0), `npm run lint` (чисто), визуальная проверка `/` в браузере на :8082 (десктоп + мобильная ширина), переключение RU/EN, работа CTA, анимации счётчиков/скролла.

## Вне рамок (YAGNI)
Глобальный `ru`-locale для дашборда; реальная лента новостей из API (статичный контент); видео-фон (используем webp); серверный рендер данных лендинга.
